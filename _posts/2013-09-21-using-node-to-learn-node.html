--- layout: post title: 'Using node to learn node' comments: true categories: [] tags: - node - nodejs status: publish type: post published: true meta: _edit_last: '1' --- I've been working on improving my NodeJS knowledge and ran across an interesting way to do that today. This is an application that teaches you some Node by giving you a set of lessons to write and learn from. It evaluates the output of each lesson to determine whether or not you did the exercise correctly I generally do not install globally so if you do not install it globally you can still verify your output by using a relative path to the verify script like this 'node ../learnyounode.js verify myprogram.js'. https://github.com/rvagg/learnyounode
