--- layout: post title: spam handling on my blog helped by using Akismet categories: [] tags: [] status: publish type: post published: true meta: {} --- Since starting this blog I have had to keep moderation enabled for comments due to the significant number of spam comments that appear each day. I have not had to moderate a single comment since enabling it. - Posted using BlogPress from my iPad
