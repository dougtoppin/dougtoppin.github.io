--- layout: post title: Using iTunes Smart Playlists for your recent podcasts categories: [] tags: [] status: publish type: post published: true meta: _edit_last: '1' --- I regularly use my iPod to listen to a number of podcasts and have been looking for an easy way to be able to play only the recent ones. I tried using the iTunes Smart Playlists feature and it appears to do what I want without me having to manually find them. The Smart Playlist allows a rule to be created which defines what you want in the playlist. The rule that I have defined includes anything that has not been played and was added in the last 7 days.
